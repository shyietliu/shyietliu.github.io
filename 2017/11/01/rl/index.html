Reinforcement Learning 1 - Markov Decision Process & Value function

发表于 2017-11-01 | 分类于 reinforcement learning

需要的背景知识有：概率论, Markov Chain, Hidden Markov Model

1.Markov Decision Process(MDP)

Recall the physics knowledge we learned in high school, you need kinematics equation to describe the free fall process. Kinematics equation help us define and depict the problem so that we could sovle it. Markov Decision Process (MDP) can depict a general reinforcement learning problem. A MDP is a tuple <S,P,R,A,γ><S,P,R,\mathcal{A},\gamma><S,P,R,A,γ>, where

S:S:S: a set of finite states

A\mathcal{A}A: a finite set of actions

P:P:P: Transition matrix. Pss′=Pr{St+1=s′∣St=s}P_{ss'}=Pr\{S_{t+1}=s'|S_t=s\}P​ss​′​​​​=Pr{S​t+1​​=s​′​​∣S​t​​=s}, s′s's​′​​ is the successor state of sss.

R:R:R: Reward function. Rsa=E{rt+1∣St=s,At=a}R_s^a=E\{r_{t+1}|S_t=s,A_t=a\}R​s​a​​=E{r​t+1​​∣S​t​​=s,A​t​​=a}.

γ:\gamma:γ: Discount rate.

Reward是在给定ttt时刻状态sss下, 对该时刻的Return的期望。γ\gammaγ是一个大于0小于1的数值，是用来平衡即时回报和长期回报的一个系数。

1.1 Return

在ttt时刻的Return (GtG_tG​t​​)被定义为从下一时刻开始，所有时刻的reward之和。

Gt=rt+1+γrt+2...=∑k=0∞γkrt+k+1G_t=r_{t+1}+\gamma r_{t+2}...=\sum_{k=0}^\infty \gamma^kr_{t+k+1} G​t​​=r​t+1​​+γr​t+2​​...=​k=0​∑​∞​​γ​k​​r​t+k+1​​

Note: In some literatures R\mathcal{R}R is employed to indicate reward function，and RRR means the return.

有时Return和Reward的符号会不容易区分，但利用下标即可分辨：Return是需要时间信息的，因为它表示从下一时刻开始所有reward的和，所以下标是时间；Reward是定义在状态sss上的，所以下标是状态。

1.2 Policy

A Policy π\piπ is a distribution over action given states. Also can be defined as a mapping from s∈Ss\in Ss∈S, a∈Aa\in \mathcal{A}a∈A to the probability π\piπ of a taking action aaa at state sss.

π(s,a)  or  π(s∣a)\pi(s,a) \ \ \text{or} \ \ \pi(s|a) π(s,a)  or  π(s∣a)

Policy 和 Action的关系

policy π\piπ 指的是在给定状态sss下采取行为aaa的概率

2. An example of MDP

下图是一个MDP的例子(From David Silver), 图中的圆形表示状态，方形表示终止状态，箭头表示状态的转移，红色字体表示可选的action，蓝色字体表示状态名称。

这个MDP描述了一个学生的生活：假设我们从class1这个状态开始，这个学生在class1的时候可以选择去玩facebook (红色的facebook表示选择)，那么他就会到达’play facebook’这个状态，在这个状态他可以一直玩下去（一直选择facebook这个action），也可选择退出（选择Quit这个action）。如果选择退出他有回到了class1这个状态，继续听课。听完课他可能会选择继续学习（Study action）这样状态就变成了class2：他去上了第二节课。第二节课也许他会睡着（sleep action），也许会继续学习（study action）进而去学第三门课class3. 以此类推。

当他在class3状态时，如果他学high了可能考虑去酒吧（pub action），去了酒吧放松之后有不同的几率回到不同的状态中。

3 Value function

MDP描述了一个RL的问题，我们用Return和Reward来表示基于某一时刻或某一状态收益。但是比如在class1状态中，这个学生可以有不同的选择，我们怎么表示在这个class1这个状态能获得的收益呢？解决办法是引入Value function.

3.1 State-Value function

Vπ(s)=E{Rt∣St=s}V^\pi(s) = E\{R_t|S_t=s\} V​π​​(s)=E{R​t​​∣S​t​​=s}

表示在状态sss, policy π\piπ 下能够获得的Return的期望。

\begin{equation} \begin{aligned} V^\pi{s} &=E_\pi\{\sum_{k=0}^\infty \gamma^k r_{t+k+1} |S_t=s\}\\ &=E_\pi\{r_{t+1}+\gamma G_{t+1}|S_t=s\}\\ &=E_\pi\{r_{t+1}+\gamma E\{G_{t+1}\}|S_t=s\}\\ &=E_\pi\{r_{t+1}+\gamma E\{G_{t+1}|S_{t+1}=s'\}|S_t=s\}\\ &=E_\pi\{r_{t+1}+\gamma V^\pi(s_{t+1})|S_t=s\}\\ &=E_\pi\{r_{t+1}|S_t=s\}+\gamma E\{V^\pi(s')|S_t=s\}\\ \end{aligned} \end{equation}

上式中
第一个等号右边是State-Value function的定义；
第二个等号右边利用Return的定义进行替换；
第三个等号右边将t+1t+1t+1时刻的reward单独提出来，其他的合并成一项即为discount后的t+1时刻的Return；
第四个等号右边利用期望运算，E{X}+E{Y}=E{X+Y}=E{X+E{Y}}=E{X}+E{Y}E\{X\}+E\{Y\}=E\{X+Y\}=E\{X+E\{Y\}\}=E\{X\}+E\{Y\}E{X}+E{Y}=E{X+Y}=E{X+E{Y}}=E{X}+E{Y}.
第五个等号右边law of total expectation： E{X}=E{E(X∣Y)}E\{X\}=E\{E(X|Y)\}E{X}=E{E(X∣Y)}
第六个等号右边根据Value function的定义代换
将所得的结果分开书写即可得到第七个等号右边的两项。下面分别对二者进行推导

E{rt+1∣St=s}E\{r_{t+1}|S_t=s\}E{r​t+1​​∣S​t​​=s}表示在sss状态下，下一时刻t+1t+1t+1的reward的期望,恰好是reward的定义，然而，这里的期望是对policy π\piπ的期望，所以我们将这个期望展开

\begin{equation} \begin{aligned} E_\pi\{r_{t+1}|S_t=s\} &=E_\pi\{R_s\}\\ &=\sum_a \pi(s,a)R_{s}^a\\ &=\sum_a \pi(s,a)\sum_{s'}P_{ss'}^aR_{ss'}^a\\ \end{aligned} \end{equation}

Eπ{Vπ(s′)∣St=s}E_\pi\{V^\pi(s')|S_t=s\}E​π​​{V​π​​(s​′​​)∣S​t​​=s}表示给定当前时刻的状态，下一个状态的State-Value function的期望

\begin{equation} \begin{aligned} E_\pi\{V^\pi(s')|S_t=s\} &=\sum_a \pi(s,a)\sum_{s'}P_{ss'}^aV^\pi(s')\\ \end{aligned} \end{equation}

下面合并两个期望，注意第二个期望要加上一个discount系数

\begin{equation*} \begin{aligned} V^\pi(s) &=E_\pi\{G_t|S_t=s\}\\ &=\sum_a \pi(s,a)\sum_{s'}P_{ss'}^aR_{ss'}^a+\gamma\sum_a \pi(s,a)\sum_{s'}P_{ss'}^aV^\pi(s')\ \ \ \ \ \ \ \ \ \ （1）\\ &=\sum_a \pi(s,a)\sum_{s'}P_{ss'}^a(R_{ss'}^a+\gamma V^\pi(s')) \end{aligned} \end{equation*}

至此，便得到了相邻状态之间State-Value function的关系。上(1)(1)(1)式中的第一项，可以看作是immediate reward, 也就是状态转移后一定会获得的一个固定的reward. 第二项可以看作长期回报，也就是新状态未来回报的期望。

3.2 Action-Value function

注意到State-Value function并没有对action进行描述，所以考虑Action-Value function
