Support Vector Machine

*This post is just a note of Andrew Ng’s lectures of SVM.

首先是notation

\boldsymbol{x}^{(i)} denotes the ithi^{th}i​th​​ training sample, y(i)∈{−1,+1}y^{(i)} \in \{-1, +1\}y​(i)​​∈{−1,+1} is the corresponding label.

Margins

SVM的目标是maximize the margin. 那么什么是margin呢？

Functional Margin

\hat{\gamma}^{(i)} = y^{(i)}(\boldsymbol{w}^T\boldsymbol{x}^{(i)}+b)

Basically, functional margin is the label multiplies the model output.

So if functional margin is positive, classified correctly. We also define the functional margin of a training set as the minimal margin over the set.

γ^=mini(γ^(i))\hat{\gamma} = min_i(\hat{\gamma}^{(i)}) ​γ​^​​=min​i​​(​γ​^​​​(i)​​)

One problem we may encounter is when we maximize the functional margin, we can always get larger margin if we scale the parameters synchronously.

\boldsymbol{w}\rightarrow K\boldsymbol{w},b\rightarrow Kb

Geometric Margin

Normalized functional margin

\gamma^{(i)}=y^{(i)}(\frac{ \boldsymbol{w}^T}{||\boldsymbol{w}||}\boldsymbol{x}^{(i)}+\frac{b}{||\boldsymbol{w}||})

Learning

Our objective is maximizing the geometric margin to get a hyperplane that separates the training data.

Version 1

SVM的学习问题就可以转化成一个有约束的优化问题：

\begin{equation} \begin{split} &\max_{\gamma,\boldsymbol{w},b} \ \gamma \\ s.t.\ \ &y^{(i)}(\boldsymbol{w}^T \boldsymbol{x}^{(i)}+b) \geqslant \gamma, \ \ \ \ \text{for all} \ i \\ \ \ &||\boldsymbol{w}||=1 \end{split} \end{equation}

也就是找到一组\gamma, \boldsymbol{w}, b, 使Geometric margin γ\gammaγ 最大. 约束条件有两个，第一是所有的样本和超平面的距离都大于等于几何间隔；第二是权重的范数为1，这可以使得几何间隔等于函数间隔。但由于有||\boldsymbol{w}||=1这一项，这样定义的优化问题并不是凸(convex)的，所以很难求解。

Version 2

为了将上述非凸（non-convex）优化问题转化成凸优化问题，我们设计一个等价的优化问题

\begin{equation} \begin{split} &\max_{\hat{\gamma},\boldsymbol{w},b} \ \frac{\hat{\gamma}}{||\boldsymbol{w}||} \\ s.t.\ \ &y^{(i)}(\boldsymbol{w}^T \boldsymbol{x}^{(i)}+b) \geqslant \hat{\gamma}, \ \ \ \ \text{for all} \ i \\ \end{split} \end{equation}

这个形式的优化问题，本质上还是最大化几何间隔，只是利用几何间隔和函数间隔的关系得到了新的形式。注意此时已经没有对权重的约束了。但是最大化项仍然是非凸的。

Version 3

我们继续加入新的Constraint，令γ^=1\hat{\gamma}=1​γ​^​​=1, 带入新的约束进上面的第二个形式的约束问题，得到

\begin{equation} \begin{split} &\max_{\boldsymbol{w},b} \ \frac{1}{||\boldsymbol{w}||} \\ s.t.\ \ &y^{(i)}(\boldsymbol{w}^T \boldsymbol{x}^{(i)}+b) \geqslant 1, \ \ \ \ \text{for all} \ i \\ \end{split} \end{equation}

进一步地，最大化$\ \frac{1}{||\boldsymbol{w}||} $ 等价于最小化$$||\boldsymbol{w}||$$, 等价于最小化$$||\boldsymbol{w}||^2$$. 所以考虑参数空间(parameter sapce)， 实际上就是quadratic的。每次一个学习一个训练样本后，原本的空间的一半都被剔除，留下符合要求的一半。

至此，该优化问题转换成了凸优化，所以可以求解。本质上是有约束的凸优化问题，基本的求解算法是拉格朗日乘子法（Lagrange Multiplier）， 高等数学中已经包含，此处不再赘述。

Lagrange Duality

在SVM中，解决原始问题的对偶（Dual）问题通常会比原始（Primal）问题更简单。那么什么是对偶问题？怎么解决对偶问题呢？

General Lagrange

这里我们给出更general的的拉格朗日法，它把等式约束拓展到了不等式约束。
现在我们不考虑SVM，仅仅考虑一个一般的有约束优化问题

\begin{equation} \begin{split} &\max_{\boldsymbol{w}} f(w) \\ s.t.\ \ \ &g_i(w) \leqslant0, i=1,2,...,k\\ &h_i(w)=0,i=1,2,...,l. \end{split} \end{equation}

为了求解上述问题，我们首先构造一个generalized Lagrangian

L(w,α,β)=f(w)+∑iαigi(w)+∑iβihi(w)\mathcal{L}(w,\alpha,\beta)=f(w)+\sum_i\alpha_ig_i(w)+\sum_i\beta_ih_i(w) L(w,α,β)=f(w)+​i​∑​​α​i​​g​i​​(w)+​i​∑​​β​i​​h​i​​(w)

这里αi, βi\alpha_i,\ \beta_iα​i​​, β​i​​是Lagrangian multipliers.
